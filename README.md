An advanced predictive modeling project identifying at-risk customers to enable proactive retention measures.

## 💡 Overview
- **Problem:** High customer acquisition costs make retention a critical business metric.
- **Solution:** A classification model computing a churn probability score per customer.
- **Techniques:** SMOTE for imbalanced data, feature scaling, and Hyperparameter tuning via Optuna.

## 🛠️ Tech Stack
- **Modeling:** XGBoost, LightGBM, Scikit-Learn
- **Data Analysis:** Pandas, Plotly
